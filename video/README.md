# AI Gateway Walkthrough Video

This folder contains the Remotion video project for the AI Gateway team walkthrough.

## Commands

- `npm install`
- `powershell -ExecutionPolicy Bypass -File .\scripts\generate-voiceover.ps1`
- `npm run studio`
- `npm run render`

## Output

The default render target is:
